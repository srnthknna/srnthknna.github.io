Linear Equation Cracker

Solves systems of linear equations by Gaussian elimination: reduce the system to triangular form, then back-substitute from the last equation up. A separate symbolic layer can also take equations written out in ordinary algebraic form and convert them into matrix rows automatically.

Solve a system

From equations to a matrix

The solver also accepts equations written symbolically — it builds a small polynomial normal form internally, reads off each variable's coefficient, and assembles the matrix row itself.

2x + 3z + 3y = 8
−2z + 3y + 2x = 3
−2y + 4x + 2z = 4
→
[2, 3, 3, 8]
[2, 3, -2, 3]
[4, -2, 2, 4]

Same system, same answer — just written the way a person would write it rather than the way the solver needs it.

Haskell implementation: row-reduction over Doubles via list recursion, plus a small multivariate-polynomial normal form (MPoly) used to read off coefficients from symbolic equations.
