A handful of things built outside of work — compilers, networking, search and game AI, and one that's just for fun. Each links to an interactive page that explains how it actually works, not just what it does.

Compiler

GVN-PRE for MLton Explore →

RIT MS project implementing GVN-PRE — a compiler optimization pass that eliminates partial redundancy MLton's existing CSE pass can't touch. Interactive subpage walks through the algorithm step by step.

Functional Programming

Linear Equation Cracker Solve →

Haskell solver for systems of linear equations via Gaussian elimination and back-substitution, plus a small symbolic layer that converts written equations into matrix form. Subpage animates the row reduction step by step.

A SAT Solver in a Few Lines of Search Search →

Boolean satisfiability solved by a tiny generic backtracking engine — the same one, unmodified, also drives the Sudoku solver below. Subpage animates the decision tree as it searches for a satisfying assignment.

Sudoku Solver via Backtracking Search Solve →

The same generic search engine as the SAT solver, just with different moves and a different goal. Subpage animates real backtracking on four puzzles — including one that lives up to its name, "diabolical."

Computer Networks

tsapp: Chained Time-Service Protocol Send a request →

A single Java program acting as client, server, or proxy for a custom time-sync protocol over raw TCP/UDP sockets, with proxies chaining together and stamping their own hop-by-hop RTT onto the reply. Subpage simulates a request traveling through a configurable proxy chain.

fcntcp: Reliable Transfer over UDP View trace →

Rebuilt TCP's reliability — checksums, sequencing, sliding-window slow start, congestion avoidance, fast retransmit — entirely on top of raw UDP. Subpage charts the real congestion window from an actual 2.96MB transfer, plus a hands-on simulator of the same rules.

Artificial Intelligence

Minimax vs. Alpha-Beta Pruning Play →

An unbeatable Tic-Tac-Toe AI in Python, implemented with both plain minimax and alpha-beta pruning. Subpage is a playable version that shows, move by move, how many states each search actually visits.

Rolling a Die Through a Maze Explore →

A* search over a puzzle where the state is position and die orientation — a cube must roll across a grid and land on the exit showing a specific face. Subpage animates the search across five mazes and three custom heuristics.
